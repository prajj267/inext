# WinSCP Setup Instructions

1. Download WinSCP: https://winscp.net/eng/download.php
2. Install and open WinSCP
3. Click "New Session"
4. Fill in these details:

   File protocol: SFTP
   Host name: 172.16.1.251
   Port: 22
   User name: [ASK YOUR SERVER ADMIN - ppraj/inext didn't work]
   Password: [YOUR PASSWORD]

5. Click "Login"
6. Navigate to the website directory (probably ~/public_html or /var/www/html)
7. Upload all files from: C:\Projects\website iit\inext-website\out\
